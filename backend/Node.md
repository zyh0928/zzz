## Tip

### Proxy

```sh
# Yarn
yarn config set proxy http://127.0.0.1:9099

yarn config set https-proxy http://127.0.0.1:9099

# NPM
npm config set proxy http://127.0.0.1:9099

npm config set https-proxy http://127.0.0.1:9099
```
