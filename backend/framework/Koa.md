## Installation

### Modules

> dependencies

```sh
yarn add @koa/cors koa koa-{router,bodyparser} uuid
```

> dev dependencies

```sh
yarn add @typs/{koa,koa-bodyparser,koa-router,uuid} cross-env nodemon ts-loader ts-node typescript -D

# old
yarn add @types/node rimraf -D
```

### TypeORM

```sh
yarn add typeorm mysql
```
