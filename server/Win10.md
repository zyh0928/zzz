## Keyboard shortcuts

### win + e

> 我的电脑

### win + r

> 运行

| 输入             | 打开         |
| ---------------- | ------------ |
| **cmd**          | 命令提示符   |
| **services.msc** | 服务         |
| **regedit**      | 注册表编辑器 |
| sysdm.cpl        | 系统属性     |
| winver           | 系统信息     |
| control          | 控制面板     |
| notepad          | 记事本       |

### win + s

> Cortana

### win + x

> 菜单

### win + i

> 设置

### win + dot

> Emoji 表情

### win + d

> 回到桌面
